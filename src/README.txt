QUADSOLVER

It solves equations that just happen to be in quadratic form. No more. No less.

Run validation inputs with:

    $ make allrun

Enter input in 'input/user_input.in' and run with:

    $ make userrun

    * Output will be in 'output/user_output.out'. *

CUnit needed for testing. Can install with: 
    sudo apt-get install libcunit1 libcunit1-doc libcunit1-dev
